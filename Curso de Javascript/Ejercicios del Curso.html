Ejercicio con do…while: duplicar, incrementar y filtrar

  1. Inicializar Arrays: Crea arrays numeros, duplicados, y resultado.
  2. Duplicar Números: Recorre numeros con un bucle do while, duplicando cada elemento y almacenándolo en duplicados.
  3. Incrementar y Filtrar: Reinicia tu índice y usa otro bucle do while para incrementar cada elemento de duplicados en 10, filtrando aquellos mayores a 15 en resultado.

Esta página muestra paso a paso cómo, a partir de un arreglo base (numeros), se construyen duplicados y luego resultado usando exclusivamente bucles do…while.

Notas
  • Se usan dos bucles do…while obligatorios: el primero para duplicar, el segundo para incrementar y filtrar.
  • El índice se reinicia entre los dos bucles.
  • La condición de filtrado es: valorDuplicado + 10 > 15.
Demostración final del código
// Inicializar arrays   
let numeros = [1, 5, 10, 20];
let duplicados = [];
let resultado = [];

// --- Duplicar Números ---
let i = 0;
do {
    duplicados.push(numeros[i] * 2);
    i++;
} while (i < numeros.length);

console.log("Duplicados:", duplicados);

// --- Incrementar y Filtrar ---
i = 0; // reiniciamos índice
do {
    let valor = duplicados[i] + 10;
    if (valor > 15) {
        resultado.push(valor);
    }
    i++;
} while (i < duplicados.length);

console.log("Resultado final:", resultado);
